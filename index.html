Descripción:

Se requiere desarrollar un sitio web que permita conocer los feriados de Chile para el año actual, para esto, se debe consumir la siguiente API. Puedes revisar la estructura con Postman y mostrar el resultado en una tabla HTML, utiliza jQuery para manipular el DOM. - Puedes usar alguna librería de CSS a tu elección si lo deseas (bootstrap, foundation, etc).

Requerimientos:

1. Cargar jQuery en el proyecto y armar una estructura básica y simple en html para el problema.

2. Crear un código para consumir la API con jQuery.

3. Utilizar JSON de respuesta y mostrar los datos en una tabla de HTML.
